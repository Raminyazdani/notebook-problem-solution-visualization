# Classification Methods Visualization

Interactive visualization and comparison of classification algorithms on synthetic datasets.

## Overview

This project demonstrates logistic regression, Linear Discriminant Analysis (LDA), and Quadratic Discriminant Analysis (QDA) for binary classification.

## Dependencies

- Python 3.7+
- NumPy
- Matplotlib
- scikit-learn
- Jupyter Notebook

## Setup

```bash
pip install -r requirements.txt
jupyter notebook
```

Open `classification_visualization.ipynb` to explore the demonstrations.
